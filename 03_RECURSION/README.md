# 03 — Recursion

Implementation and exploration of recursive algorithms.

## Concepts

* Recursion basics
* Function call stack
* Tail recursion
* Head recursion
* Linear recursion
* Tree recursion
* Indirect recursion
* Nested recursion

## Problems Implemented

* Sum of N numbers
* Factorial
* Power calculation
* Taylor series
* Fibonacci
* Fibonacci with memoization
* nCr
* Tower of Hanoi

## Goal

Develop **recursive thinking** and understand how recursion uses the program stack.
