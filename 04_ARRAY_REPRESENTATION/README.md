## 📖 Overview

Understanding how arrays are fundamentally stored in memory before interacting with them as an Abstract Data Type. 

> 💡 **Key Idea:** Arrays are stored in **contiguous memory locations**. This physical layout property is exactly what allows for **O(1) constant time access** using an index.

---

## 🎯 Goal

> *Understand **how memory is allocated** for arrays, how **element addresses are calculated** under the hood, and how **multidimensional arrays map** to linear 1D memory.*
