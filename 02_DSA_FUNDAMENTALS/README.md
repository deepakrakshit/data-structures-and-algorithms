## 📖 Overview

Core concepts used to analyze and design algorithms. Before writing complex data structures, it is vital to understand how to measure their efficiency and how data is conceptually versus physically stored in memory.

---

## 🎯 Goal

> *Understand **how algorithms behave as input size grows** and how memory is used during execution.*
